Typing Hints

接續上一則筆記，我們繼續探討Typing Hint，這是 FastAPI 特別著重的重點：

實作Typing Hint的意義其實就是:

在定義函式的參數時，連同型態一併定義

:::python # Without typing hint @app.get("/users/{user_id}) def read_user(user_id, q): # 只放變數名稱 return {"user_id": user_id, "q": q}

# With typing hint
@app.get("/users/{user_id})
def read_user(user_id: int, q: Optional[str] = None): # 除了變數名稱，還設定的變數的型態
    return {"user_id": user_id, "q": q}

實作Typing Hint的特色：

1.Data Conversion

自動將Request的字串轉換(cast)為指定的格式

以下圖的API為例:

:::python @app.get("/items/{item_id}") async def read_item(item_id: int): return {"item_id": item_id}
